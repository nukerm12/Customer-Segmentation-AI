# Customer-Segmentation-AI
This project demonstrates the application of Unsupervised Learning to analyze customer behavior. Using a marketing dataset, I implemented K-Means Clustering in Python to identify distinct customer segments.
Key Technical Highlights:

Data Processing: Cleaned and prepared raw marketing data using pandas and numpy.

Clustering Logic: Applied the Elbow Method to find the optimal number of clusters for the model.

Analysis: Interpreted demographic factors like Income and Spending Habits to generate actionable insights.

Tools: Developed in Python using scikit-learn for modeling and matplotlib/seaborn for data visualization.
